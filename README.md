# WPCharts

Приложение для построения графиков по координатам.

## Настройка проекта

### Установка

```bash
$ npm install
```

### Режим разработки

```bash
$ npm run dev
```

### Сборка

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
